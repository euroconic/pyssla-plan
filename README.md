# Pyssla Plan 🎨

A kid-friendly pattern builder for IKEA Pyssla and Hama beads.

## 🚀 Overview

Pyssla Plan is designed to bridge the gap between digital creativity and physical crafting. It focuses on a "Healthy Kid" philosophy—scaffolding the creative process without the addictive mechanics of modern games.

### Key Features

- **Magic Maker:** AI-driven voice-to-pattern generation (multilingual support).
- **Smart Pencil:** Intuitive editing tools designed for children.
- **1:1 Scale Printing:** Print patterns at exact physical size (5mm beads) to use as pegboard templates.
- **Social Metrics:** Track views, downloads, and "remixes" to build pride in creations.
- **Neuro-Developmental Focus:** Features designed to improve pincer grasp, visuospatial reasoning, and executive function.

## 📂 Documentation

Comprehensive research and strategy documents are located in the `/docs` folder:
- **Product Vision:** Pedagogical goals and neuroscience.
- **Competitor Review:** Benchmarking against Hama Universe, Perlypop, etc.
- **Growth Strategy:** Zero-budget user acquisition.

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **State Management:** React Hooks
- **Icons:** Lucide React
- **Voice:** Web Speech API

## 📦 Getting Started

1. `npm install`
2. `npm run dev`
